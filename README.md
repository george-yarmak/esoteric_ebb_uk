**Версія гри:** 1.1.3  
**Тип перекладу:** ШІ-генерація

У цьому репозиторії лежить часткова українська локалізація для RPG *Esoteric Ebb*. 
Важливе уточнення: це автоматичний, згенерований штучним інтелектом машинний переклад. Він не ідеальний, не проходив ручну літературну редактуру і чесно несе звання "ШІ-слопу". Цей мод створений виключно для того, щоб ви могли пройти гру рідною мовою вже сьогодні, не чекаючи на повноцінний ручний переклад.

1. Розпакуйте архів з локалізацією в папку з грою біля виконуючого файлу гри (наприклад, `\Steam\steamapps\common\Esoteric Ebb`).
2. Можете не робити бекап, якщо шось піде не так просто перевстановите гру, вона не така уж і важка. :)
3. Запустіть setup_uk.bat
4. Спробуйте пограти в гру
